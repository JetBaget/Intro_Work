Здравствуйте!

Высылаю тестовое задание.

Python 2.7
ОС: Linux.
URL web-сервиса: http://gptl.ru/interview_task/api/commands

Разработать программу на языке Python. Программа должна посылать GET запрос с параметром aim=python_developer_job к web-сервису. В полученном ответе (json) находится команда (одна из стандартных утилит Linux) с параметрами. Выполнить ее в отдельном процессе (использовать стандартный модуль subprocess). Основной процесс должен получить выводимые стандартной утилитой данные, используя средства модуля subprocess. Результат залогировать в файл (использовать стандартный модуль logging), расположенный в директории программы, и отправить на сервер POST запросом (command=(команда), result=(вывод команды)). Сервер может вернуть невалидную команду (wrong_command), подобную ситуацию нужно обрабатывать.

Для отправки запросов, как вариант, можно использовать библиотеку requests http://docs.python-requests.org/en/master/

Примечание Сервис не возвращает команду rm (если вдруг у Вас сомнения возникнут насчет возвращаемых команд).
